* `apt update`
* `apt install git curl php openssh -y`
* `git clone git://github.com/tecnicaljainil/tecnicaljainilinsta.git`
* `cd tecnicaljainilinsta`
#### > Run : `bash tecnicaljainilinsta.sh`
